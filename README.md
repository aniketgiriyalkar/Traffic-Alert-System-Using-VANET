# Traffic-Alert-System-Using-VANET
Traffic-Alert-System-Using-VANET is a final-year project about reporting and
sharing traffic conditions through a location-aware platform. The original stack
reflects the project’s era, so this repo is best read as a prototype and a
starting point for a more modern location-sharing app.

## Quick start

1. Inspect the client and server pieces in the repository.
2. Run the backend in the PHP/SQL environment the project expects.
3. Connect the client to the configured server endpoint and test the alert flow.

## If you modernize it later

The best 2026 rewrite would use `React Native` or `Flutter` for the client,
`FastAPI` or `NestJS` for the backend, and `PostgreSQL + PostGIS` for the map
data. That stack would give you better geospatial querying and a much cleaner
mobile experience.
